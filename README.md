# Supermarket Sales Analysis (SQL + Python + Power BI)

## Project Overview
This project analyzes supermarket sales data to uncover insights into customer behavior, product performance, and sales trends.

## Tools Used
- PostgreSQL
- Python (Pandas, Matplotlib)
- Power BI

## Key Insights
- Food & Beverages generated the highest revenue.
- Most transactions occur in the evening.
- Cash is the most common payment method.

## Project Workflow
1. Data Cleaning
2. SQL Analysis
3. Advanced SQL Window Functions
4. Python Visualization
5. Business Dashboard
